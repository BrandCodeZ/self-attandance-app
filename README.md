# Staff Attendance App

A web app to track your staff's check-in and check-out times.

## Features

- **Staff management** — Add, edit, and remove staff with name, role, and employee ID
- **Check-in / Check-out** — One-click attendance with automatic timestamps
- **Live clock** — Real-time date and time display
- **Dashboard** — See who's currently in, today's stats at a glance
- **Attendance records** — Filter by date and staff member
- **Export CSV** — Download attendance reports

## Quick Start (No install needed)

**Double-click or open in your browser:**

```
C:\Users\ARSE\staff-attendance\attendance.html
```

Works immediately — no Node.js required. Data saves in your browser.

## Developer Mode (Optional)

If you have Node.js installed:

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build for Production

```bash
npm run build
npm run preview
```

## Data Storage

All data is saved in your browser's local storage on this device. No server or account required.

## Usage

1. Go to **Staff** and add your team members
2. Use **Check In/Out** to record when staff arrive and leave
3. View **Dashboard** for a live overview
4. Use **Records** to review history and export CSV reports
